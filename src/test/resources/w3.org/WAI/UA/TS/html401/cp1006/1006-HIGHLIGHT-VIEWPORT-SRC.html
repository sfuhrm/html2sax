UAAG 1.0 Requirement

Checkpoint 10.6 Highlight current viewport (Priority 1 )
Provision 1 : Highlight the viewport with the current focus (including any frame that takes current focus).
Provision 2 : For graphical viewports, as part of satisfying provision one of this checkpoint, provide at least one highlight mechanism that does not rely on rendered text foreground and background colors alone (e.g., use a thick outline).

Test 1 :

Procedure

  1. Move content focus to each of the frames on this page.

Run test

(Test code contained in the frameset document.) Link to the test frameset

Expected results

  1. The user agent highlights the frame with current focus.
  2. The highlight mechanism used by the user agent does not rely on rendered text foreground and background colors alone (e.g., use a thick outline).
